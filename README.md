## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/aliimm/airbnb.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. CD into the frontend folder
   ```sh
   npm start
   ```
4. CD into the backend folder
   ```sh
   npm start
   ```
